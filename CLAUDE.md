# Council — Claude Code Context

## What this project is

A self-hosted multi-persona AI agent ecosystem built on Hermes Agent (NousResearch).
Each persona is an isolated Hermes profile with its own identity, model, tools, and
operating procedures. Personas share common infrastructure (PostgreSQL, vLLM, audit log).

## Two-repo structure

| Repo | Purpose |
|---|---|
| `Personas` | Shared infrastructure, persona template, deploy scripts |
| `Council` | Persona definitions (one folder per persona) |

Personas repo knows nothing about specific personas. Council depend on Personas repo.

## Key technical decisions

- **Agent framework:** Hermes Agent (NousResearch) — one Hermes profile per persona
- **Local inference:** vLLM serving Qwen2.5-14B-Instruct-GPTQ-Int4 (fits RTX 5070 12GB VRAM)
- **Primary inference:** Anthropic API (Claude Opus/Sonnet depending on persona)
- **Database:** PostgreSQL with pgvector (pgvector/pgvector:pg16 image)
- **GPU:** RTX 5070, NVIDIA container toolkit required for vLLM container
- **Persona identity files:** SOUL.md (character) + AGENTS.md (operating procedures)
- **Deploy flow:** edit persona.json → run deploy-persona.sh → HERMES_HOME=... hermes

## Commands

```bash
# Start at Personas home directory
cd /mnt/c/Projects/Personas

# Start shared infrastructure (from Personas)
cd infrastructure && docker compose up -d

# Create a new persona (from Council)
cp -r ../Personas/template personas/persona-id

# Deploy a persona (from Personas)
./scripts/deploy-persona.sh ../Council/personas/persona-id

# Start a persona
HERMES_HOME=/mnt/c/Projects/Council/personas/persona-id/.hermes hermes
```

## What not to do

- Do not commit .env files or API keys
- Do not put persona-specific content in inner-sanctum-infra
- Do not hardcode model names anywhere except persona.json and config.yaml
- Do not modify the audit_log table structure without updating the append-only trigger

## Additional Notes

- Any place 'persona-id' is mentioned, this should be substituted with the Persona's actual name (NORA, for instance)
